## Description
Skill 3: analizza autocertificazioni PA / identifica clausole di esonero sulla base delle norme citate nel documento stesso. Caso d uso reale BDSR Punto 2 (gas/CO esonerato per <4 appartamenti via comma 7 art. 13-ter DL 145/2023). Indipendente da T3-T5 (consuma solo T2) può essere lavorata in parallelo a pa-form-filler. Limite esplicito: cita solo norme presenti nel documento — no interpretazioni proprie.
