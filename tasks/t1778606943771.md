## Description
Sub-modulo di pa-form-filler che gestisce session token Keycloak (scadono dopo ~30min su portali PA). Salva progressi su disco / rileva 401/redirect login / propone resume da campo X. Sequenziale a T4 (stesso pa-form-filler/SKILL.md).

## Acceptance Criteria
- Checkpoint file schema documentato in SKILL.md: ~/.claude/pa-checkpoints/{portale}-{YYYY-MM-DD}.json
- Trigger save: ogni cambio pagina + ogni 5 campi compilati
- Detection HTTP 401 / redirect login / DOM sessione scaduta → pausa + prompt resume
- Resume esplicito invocabile dall utente (riprendi il form X) con lista checkpoint disponibili
- Cleanup automatico checkpoint dopo submit andato a buon fine
- Test bats coprono: creazione JSON con schema corretto — resume da checkpoint — simulazione 401 trigger — eliminazione post-submit
